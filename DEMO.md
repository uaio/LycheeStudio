## 💡 技术说明

### Web 版本特点
- **完整 UI**: 所有界面组件都可正常交互
- **响应式设计**: 支持不同屏幕尺寸
- **现代样式**: 渐变色彩和流畅动画
- **类型安全**: TypeScript 完整支持

### 桌面版要求
要运行完整的桌面版，需要：
```bash
# 1. 安装 Rust（网络允许时）
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
rustup default stable

# 2. 运行桌面应用
npm run tauri:dev
```
